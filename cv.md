## Code example
### Classical FizzBuzz game

``` js
const fizzOrBuzz = (number) => {
    let res = "";
    if (number % 3 === 0) res+="Fizz";
    if (number % 5 === 0) res+="Buzz";
    if (res === "") res = number;
    return res;
}

const fizzBuzzLoop = (loopEnd) => {
    for (i = 1; i<=loopEnd; i++) {
        fizzOrBuzz(i)
    }
}
```
